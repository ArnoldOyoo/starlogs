# StarLogs

## Project info

StarLogs is a Vite + React + TypeScript application.

## Local development

Prerequisite: Node.js and npm installed.

```sh
# 1) Install dependencies
npm install

# 2) Start the dev server
npm run dev

# 3) Build for production
npm run build
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
